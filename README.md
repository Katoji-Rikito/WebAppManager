# Ứng dụng quản lý trên nền web (WebAppManager)

Ứng dụng web dành cho việc quản lý các loại như chi tiêu, danh bạ, game,...

Sử dụng ASP.NET Core 8, MariaDB bản mới nhất và triển khai trên Docker!!!
